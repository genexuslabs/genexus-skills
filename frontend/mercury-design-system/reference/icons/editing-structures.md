# Editing Structures

Icons for data structure creation and modification, representing attributes, formulas, keys, and indexes

**Type:** Multicolor

## Usage

```ts
import { getIconPath } from "@genexus/mercury/assets-manager.js";

getIconPath({ category: "editing-structures", name: "attribute-key" });
```
