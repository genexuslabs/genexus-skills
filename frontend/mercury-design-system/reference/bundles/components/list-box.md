# List Box

Bundle name: `"components/list-box"`

Implementation: [list-box.css](./list-box.css)

## Classes

### `list-box`

Applies to the components: `ch-action-list-render`

A List Box component is a versatile UI element used to present a selectable list of items, offering users a straightforward way to make choices or navigate through options. Supports single and multiple selection, icons, separators, groups, disabled items, and additional information display. Commonly used in forms, dropdown menus, and selection interfaces
