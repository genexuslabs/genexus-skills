# Sidebar

Bundle name: `"components/sidebar"`

Implementation: [sidebar.css](./sidebar.css)

## Classes

### `sidebar`

Applies to the components: `ch-sidebar`

A sidebar component that provides a collapsible side panel for navigation or auxiliary content. Features a configurable collapsed width, expand/collapse button, and spacing body tokens for padding

### `sidebar-collapsed-hidden`

Applies to the components: `ch-sidebar`

Variant of sidebar that fully hides the sidebar when collapsed (sets collapsed inline size to 0), rather than showing a narrow strip

### `sidebar-with-motion`

Applies to the components: `ch-sidebar`

Variant of sidebar that adds smooth transition animations when expanding or collapsing. Applies ease timing transitions to the sidebar width and the expand button rotation
