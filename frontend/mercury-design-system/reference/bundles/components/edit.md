# Edit

Bundle name: `"components/edit"`

Implementation: [edit.css](./edit.css)

## Classes

### `input`

Applies to the components: `ch-edit`

A primary HTML input element designed to capture user data and facilitate interactions. It serves as the main interface for users to enter information such as text, numbers, or dates. Supports various states including disabled, read-only, with icons, and placeholder text

### `input-error`

Applies to the components: `ch-edit`

Error state variant for input components. Applied to inputs that have validation errors or invalid input, providing visual feedback to users
