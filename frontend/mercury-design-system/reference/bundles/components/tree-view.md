# Tree View

Bundle name: `"components/tree-view"`

Implementation: [tree-view.css](./tree-view.css)

## Classes

### `checkbox`

Applies to the components: Checkbox within `ch-tree-view-render` items

Checkbox styling included within the tree-view bundle for tree items with checkboxes. When the `checkbox` and `toggleCheckboxes` properties are enabled on the tree view, each item displays a checkbox. See checkbox component documentation for details

### `tree-view`

Applies to the components: `ch-tree-view-render`

A hierarchical UI element that visually represents a nested data structure, allowing users to expand and collapse branches. Commonly used for navigating directories, organizing data, and managing complex relationships. Supports show-lines modes (`last`, `all`), node icons, drag-and-drop reordering, and inline checkboxes for multi-selection
