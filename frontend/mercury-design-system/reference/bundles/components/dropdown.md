# Dropdown

Bundle name: `"components/dropdown"`

Implementation: [dropdown.css](./dropdown.css)

## Classes

### `button-icon-and-text`

Applies to the components: `button`, `a`

Used in combination with button variant classes within dropdown trigger buttons. See button component documentation for details

### `button-icon-only`

Applies to the components: `button`, `a`

Used in combination with button variant classes within dropdown trigger buttons. See button component documentation for details

### `button-primary`

Applies to the components: `button`, `a`

Primary button styling for dropdown trigger buttons. See button component documentation for details

### `button-secondary`

Applies to the components: `button`, `a`

Secondary button styling for dropdown trigger buttons. See button component documentation for details

### `button-tertiary`

Applies to the components: `button`, `a`

Tertiary button styling for dropdown trigger buttons. See button component documentation for details

### `dropdown`

Applies to the components: `ch-dropdown`

A dropdown component that displays an action menu with various options. Supports nested items, icons, separators, disabled states, custom content, and header/footer slots. The menu appears when the trigger button is activated

### `footer-slot`

Applies to the components: Slotted footer content in `ch-dropdown`

Styles applied to custom footer content slotted into the dropdown component

### `header-slot`

Applies to the components: Slotted header content in `ch-dropdown`

Styles applied to custom header content slotted into the dropdown component
