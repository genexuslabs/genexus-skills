# Checkbox

Bundle name: `"components/checkbox"`

Implementation: [checkbox.css](./checkbox.css)

## Classes

### `checkbox`

Applies to the components: `ch-checkbox`

A checkbox input element is a versatile component designed to allow users to make binary choices. It supports multiple states including checked, unchecked, indeterminate, and disabled. Can be used with icons and supports both block and inline grouping layouts

### `checkbox-error`

Applies to the components: `ch-checkbox`

Error state variant for checkbox components. Applied to checkboxes that have validation errors or invalid input, providing visual feedback to users
