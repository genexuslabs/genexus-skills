# Pills

Bundle name: `"components/pills"`

Implementation: [pills.css](./pills.css)

## Classes

### `pill`

Applies to the components: `ch-combo-box-render` (specifically configured for pills)

A pill is a compact, rounded UI element used to display labels, categories, status indicators, or selections. It visually stands out with a subtle, often colorful background and status icons. Pills can be interactive, allowing users to select or change them. Supports different states: enabled, processing, success, error, warning, and information

### `pills`

Applies to the components: `ch-combo-box-render` (specifically configured for pills)

Alternative class name for the pill component. See `pill` class documentation

### `pills-container`

Applies to the components: Container elements (e.g., `div`)

Container class for grouping multiple pill components. Provides flex layout with appropriate wrapping and spacing between pills
