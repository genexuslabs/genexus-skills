# Combo Box

Bundle name: `"components/combo-box"`

Implementation: [combo-box.css](./combo-box.css)

## Classes

### `combo-box`

Applies to the components: `ch-combo-box-render`

A combo box is a versatile UI component that combines the features of a text input and a select element. It allows users to either type a custom value or select from a predefined list of options. This component enhances user interaction by providing auto-suggestions, search functionality, and the flexibility to handle both fixed and dynamic input

### `combo-box-error`

Applies to the components: `ch-combo-box-render`

Error state variant for combo box components. Applied to combo boxes that have validation errors or invalid input, providing visual feedback to users
