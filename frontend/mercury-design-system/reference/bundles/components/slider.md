# Slider

Bundle name: `"components/slider"`

Implementation: [slider.css](./slider.css)

## Classes

### `slider`

Applies to the components: `ch-slider`

A slider is a versatile component used to allow users to select a value or range within a predefined spectrum. It is designed to be visually prominent and intuitive, often serving as a control for adjusting settings like volume, brightness, or price range. Tip: Apply `display: grid` to the slider's container to make the slider stretch to 100% width
