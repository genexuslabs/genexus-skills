# Elevation

Bundle name: `"utils/elevation"`

Implementation: [elevation.css](./elevation.css)

## Classes

### `elevation-1`

Applies to the components: Any container element (e.g., `div`)

Applies the first level of elevation background color. Use when the container has no ancestor with an elevation background. This maintains a clear visual hierarchy by enhancing contrast between nested elevation containers

### `elevation-2`

Applies to the components: Any container element (e.g., `div`)

Applies the second level of elevation background color. Use when the parent container has an `elevation-1` background. This maintains a clear visual hierarchy by enhancing contrast between nested elevation containers

### `elevation-3`

Applies to the components: Any container element (e.g., `div`)

Applies the third level of elevation background color. Use when the parent container has an `elevation-2` background. This maintains a clear visual hierarchy by enhancing contrast between nested elevation containers
