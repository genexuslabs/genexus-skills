# GX Server

Server collaboration icons representing version control, commits, locks, and repository management

**Type:** Multicolor

## Usage

```ts
import { getIconPath } from "@genexus/mercury/assets-manager.js";

getIconPath({ category: "gx-server", name: "conflict" });
```
