# GX Test

Testing framework icons for unit tests, UI tests, test suites, and result reporting

**Type:** Multicolor

## Usage

```ts
import { getIconPath } from "@genexus/mercury/assets-manager.js";

getIconPath({ category: "gx-test", name: "run-tests" });
```
