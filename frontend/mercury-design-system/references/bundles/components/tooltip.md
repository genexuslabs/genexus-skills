# Tooltip

Bundle name: `"components/tooltip"`

Implementation: [tooltip.css](./tooltip.css)

## Classes

### `tooltip`

Applies to the components: `ch-tooltip`

A tooltip is a small, interactive pop-up that appears when a user hovers over or focuses on an element. It provides additional information or context without cluttering the interface. Supports multiple positioning options via blockAlign (outside-start, outside-end) and inlineAlign (center, inside-start, inside-end)
