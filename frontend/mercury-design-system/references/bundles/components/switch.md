# Switch

Bundle name: `"components/switch"`

Implementation: [switch.css](./switch.css)

## Classes

### `switch-large`

Applies to the components: `ch-switch`

Large variant of the switch (toggle) component. A switch allows users to toggle between two states, such as on/off or enabled/disabled. The large size provides a bigger touch target with checked/unchecked captions. Supports disabled state

### `switch-small`

Applies to the components: `ch-switch`

Small variant of the switch (toggle) component. A compact switch for use in tighter layouts. Supports the same states as the large variant (checked, unchecked, disabled) with checked/unchecked captions
