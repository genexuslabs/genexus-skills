# Segmented Control

Bundle name: `"components/segmented-control"`

Implementation: [segmented-control.css](./segmented-control.css)

## Classes

### `segmented-control-filled`

Applies to the components: `ch-segmented-control-render`

A segmented control is a linear set of two or more segments, each of which functions as a button. The filled variant features a solid background for the selected segment, providing strong visual distinction. Supports multiple elevation levels and disabled states

### `segmented-control-item`

Applies to the components: Individual items within `ch-segmented-control-render`

Styles individual segmented control items (segments). Each item can be in a selected, unselected, or disabled state. Uses CSS parts (action, selected, unselected) for fine-grained styling of different states
