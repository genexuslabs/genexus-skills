# MODULE_OBJ Properties

## Module Manager Information

- `PackagedModuleName` (Text) — Package Name
- `ModuleVersion` (Text) — ModuleVersion. Version. Default: `1.0`
- `ModuleDescription` (Text) — ModuleDescription. Description
- `ModuleAuthor` (Text) — ModuleAuthor. Author
- `ModuleLicenseUrl` (Text) — Module License URL. URL to the document defining how and when this module can be used. Note that the license apply directly to this module and not the list of dependencies.
- `ModuleProjectUrl` (Text) — Module Project URL. URL to the document defining the details about this module. Try to give a good description in terms of what is this project about and how to use it right.
- `ModuleServerUrl` (Text) — Module Server URL. URL to GeneXus Server associated to this project
- `ModuleTags` (Text) — ModuleTags. Comma separated values of tags. Tags will help to find this module
- `ModuleOwner` (Text) — ModuleOwner. Owner
- `ModuleResources` (Custom) — ModuleResources. List of File objects defined as resources for the module (e.g. External Object implementation); they are needed to integrate the packaged module in another Knowledge Base.
