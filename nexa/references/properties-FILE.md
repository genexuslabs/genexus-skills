# FILE Properties

## (root)

- `ObjName` (Text) — Name
- `ObjDesc` (Text) — Description. Describe the content or purpose of the element

## (root)

- `FileExternalFileName` (Text) — External File Name. Name of the file when it is extracted from the Knowledge Base to the file system.
- `JavaExtract` (Boolean) — Extract for Java Generator
- `JavaExtractFolder` (Custom) — Java Generator Extraction Directory [hidden]
- `NetExtract` (Boolean) — Extract for .Net Generator
- `NetExtractFolder` (Custom) — .Net Generator Extraction Directory [hidden]
- `NetCoreExtract` (Boolean) — Extract for .Net Core Generator
- `NetCoreExtractFolder` (Custom) — .Net Core Generator Extraction Directory [hidden]
- `IOSExtract` (Boolean) — Extract for iOS Generator
- `iOSExtractFolder` (Custom) — iOS Generator Extraction Directory [hidden]
- `AndroidExtract` (Boolean) — Extract for Android Generator
- `AndroidExtractFolder` (Custom) — Android Generator Extraction Directory [hidden]
- `ExtractZip` (Boolean) — Extract Zip. Extract zip file content when copying
