# IDX Properties

## (root)

- `HelpKeyword` (Integer) — HelpKeyword [hidden]
- `IDXSTOAREA` (Text) — Storage area
- `IDXINISIZE` (Integer) — Initial size
- `IDXFEXTENT` (Integer) — First extent size
- `IDXNEXTENT` (Integer) — Next extents percentage increase
- `IDXMINEXTENT` (Integer) — Minimum number of extents
