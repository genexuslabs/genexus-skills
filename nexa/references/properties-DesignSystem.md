# DesignSystem Properties

## (root)

- `ObjName` (Text) — Name
- `ObjDesc` (Text) — Description. Describe the content or purpose of the element

## (root)

- `Folder` (Custom) — Folder
- `BaseCssLibrary` (Combo) — Base CSS. A CSS library to include as a base style.. Default: `idNone`. Values: None, Bootstrap v3, Bootstrap v3 RTL
- `DefaultTemplateLayout` (Combo) — Default Template Layout. Specifies a template used to generate default Transactions and WorkWith web layouts.. Default: `UnanimoTemplate`. Values: FlatTemplate, UnanimoTemplate, CarmineTemplate, FioriTemplate
